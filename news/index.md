# Changelog

## timesfm 0.1.0

- Initial release.
- [`timesfm()`](https://mattyoreilly.github.io/timesfm/reference/timesfm.md)
  produces zero-shot point and decile forecasts of univariate time
  series using Google Research’s TimesFM 2.5 (200M) foundation model.
- [`install_timesfm()`](https://mattyoreilly.github.io/timesfm/reference/install_timesfm.md)
  performs one-time setup of the underlying Python package into a
  dedicated virtualenv, requiring Python \>= 3.10.
