# timesfm: R Interface to 'TimesFM', a Foundation Model for Time Series Forecasting

A thin wrapper around the 'TimesFM' 'Python' package from Google
Research (<https://github.com/google-research/timesfm>). Provides
zero-shot probabilistic forecasting of univariate time series from a
pretrained foundation model, with no training or tuning.
